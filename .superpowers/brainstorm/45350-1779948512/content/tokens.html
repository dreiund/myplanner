How tokens.css feeds both CSS and Naive UI
tokens.css — loaded first
:root {
  --app-blue: #0066cc;
  --app-ink: #1d1d1f;
  --app-canvas: #ffffff;
  --app-parchment: #f5f5f7;
  ...
}
→
global.css — typography baseline
body {
  font-family: "SF Pro Text"...;
  font-size: 17px;
  color: var(--app-ink);
  background: var(--app-parchment);
}
→
AppLayout.vue — Naive UI theme
themeOverrides: {
  common: {
    primaryColor:
      '#0066cc',
    borderRadius:
      '18px',
  }
  Button: { ... }
}
